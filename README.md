## Основные команды для работы 
#### Инициализация проекта:
```
npm i
```

#### Запуск линтера:
```
npm run lint
```

#### Запуск линтера в режиме исправления и форматтера кода:
```
npm run fix
```

#### Запуск dev сервера на 3000 порту:
```
npm run dev
```
